RPM および DNF パッケージ管理について学習します。このモジュールでは、RPM パッケージのクエリ、RPM を使用したインストールと削除、DNF を使用したパッケージ管理、DNF の履歴と元に戻す操作、ローカルリポジトリの作成について扱います。実践的なラボと実世界の課題を通じて、これらの不可欠な Linux スキルを習得します。
